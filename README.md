    - SQL資料庫串接：會員資料、訂單與明細、優惠券。  
    - Node.js-RESTful API 路由設計：會員、訂單與明細、優惠券、登入驗證、Google登入。  
    - Figma 視覺規劃：會員相關頁面。  
    - 建立MySQL資料庫：會員資料、訂單與明細、優惠券。  
    - 企劃書撰寫：網站視覺規劃、會員相關頁面介紹、企劃書規格統整、封面製作。
